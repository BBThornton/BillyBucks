# Starting the Demo Site
To start the demo site you can run the command
> npm start dev

This will require Node and npm to be installed.

It will also be necessary to ensure you are running the webserver manager from the relevant repository. 

You will also need to change the web socket url in the demo site mine.html file.

In the start script the following url will need to changed
> server = "ws://172.17.135.209:8181";
